## Project Overview
CreditWise is a Machine Learning-based web application that predicts whether a loan application is likely to be approved based on applicant details such as income, credit history, and financial information.

The project demonstrates an end-to-end ML pipeline including data preprocessing, model training, and deployment using Streamlit.

---

## Features
- Data preprocessing and feature engineering
- Categorical encoding and feature scaling
- Machine Learning prediction model
- Interactive Streamlit web interface
- Real-time loan approval prediction
